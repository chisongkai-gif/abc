# 《寻找就近养老院》App 全面交互设计原型

## 运行方式
解压后双击 `index.html` 即可运行，无需安装环境。

## 本版本新增内容
相比前一版，本版本更适合毕业设计答辩演示，加入了完整产品闭环和更多交互细节。

### 一、用户端完整流程
启动页 → 角色选择 → 引导页 → 登录页 → 首页 → 搜索/筛选 → 附近养老院 → 地图找院 → 详情页 → 收藏/对比/在线咨询 → 预约参观 → 预约成功 → 我的预约 → 服务评价。

### 二、服务闭环
用户可以完成：
- 查找养老院
- 筛选养老院
- 查看资质、收费、床位、服务项目
- 收藏机构
- 机构对比
- 在线咨询
- 预约参观
- 查看预约记录
- 提交评价

### 三、适老化设计
加入：
- 长辈模式
- 高对比显示
- 语音播报开关示意
- 一键求助
- 大按钮、大字号、少层级交互

### 四、机构端
加入机构端演示：
- 机构信息管理
- 预约订单管理
- 床位更新
- 服务套餐发布
- 机构入驻申请

### 五、适合写进论文/答辩的说明
本原型体现了“线上查询平台 + 线下服务对接”的双向服务体系，通过定位、筛选、详情展示、预约、评价等流程，构建从信息获取到服务确认的完整交互闭环。
